MCP Connector Information

Use this information to set up the connection in Le Chat

Connection URL

https://mcp-look-4-fundings-5135f019.alpic.live/

Setup Instructions

  1. 1 Open Le Chat in your browser
  2. 2 Navigate to Intelligence → Connectors
  3. 3 Click "Add Connector"
  4. 4 Select "Custom MCP Connector"
  5. 5 Choose a name (e.g., "Look 4 Fundings") and paste the connection URL above
  6. 6 Click "Connect" to establish the connection
