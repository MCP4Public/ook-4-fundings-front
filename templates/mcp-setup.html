MCP Functions

Explore the available MCP (Model Context Protocol) functions for integrating with the Look 4 Fundings platform.

Search Funding

Search for EU funding opportunities by keyword and return a list of funding details.

Get Company Profile

Retrieve the company profile information from the Look 4 Fundings web application to provide context for better funding recommendations.

Get All Grants

Retrieve all grants currently listed in the Look 4 Fundings web application, including details like title, URL, summary, deadline, status, budget, and company affinity.

Update Company Scope

Update only the company scope in the Look 4 Fundings web application by fetching the current profile, updating the scope field, and saving the changes.

Generate PDF Report

Generate a professional PDF report for the company. Creates reports from company profile and won grants data, or custom reports from provided text.

Add New Grant

Add a new grant to the Look 4 Fundings web application with a specified affinity score by updating the PublicFunding object and creating a new entry.
